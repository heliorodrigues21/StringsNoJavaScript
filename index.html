Tipos de dados no JS

Strings

Dados do tipo String são entendidos como sendo cadeias de caracteres. Esse conceito aplica-se não apenas ao JS, mas também em outras linguagens de programação, como por exemplo o Java.

Por padrão existem 3 formas de imprimir String no JS através do objeto console.log()

* Utilizando aspas duplas console.log("Imprimindo com Aspas Duplas");

*Utilizando aspas simples console.log('Imprimindo com Aspas Simples')

* Utilizando a Template literals console.log(`Template literals`)

A vantagem ao utilizar a Template Literals é que podemos utilizar o recurso de multilinha console.log(`Imprimindo
o texto
em mais de uma linha`)

A Template Literals também permite que possamos colocar expressões dentro da própria String console.log(`Soma 2+2 = ${2+2}`)
